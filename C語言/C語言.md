```
1.%c：以字元方式輸出
2.%d：10 進位整數輸出
3.%o：以 8 進位整數方式輸出
4.%u：無號整數輸出
5.%x、%X：將整數以 16 進位方式輸出
6.%f：浮點數輸出
7.%e、%E：使用科學記號顯示浮點數
8.%g、%G：浮點數輸出，取 %f 或 %e（%f 或 %E），看哪個表示精簡
9.%%：顯示 %
10.%s：字串輸出
11.%lu：long unsigned 型態的整數
12.%p：指標型態
```
