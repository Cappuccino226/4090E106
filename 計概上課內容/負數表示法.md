### 負數的表示法
```
1.帶符號大小表示法: 在最左邊留一個位元來表示正負號

2.1's補數表示法: 將每個位元反向

3.2's補數表示法: 算成二進位數之後+1
```

### 1's補數[complement]
```
step1:判斷正負

step2:計算大小(算出二進位數)

  2-1:執行反向運算
  
step3:完成 [正負號] + [大小]
```

### 2's補數[complement]
```
step1:判斷正負

step2:計算大小(算出二進位數)

  2-1:計算完+1
  
```
